# Scribble_Stack
Don't be a Scrub. Be a Scrib. <br>
Doodly scribbly little game

# How to play
Scribble the word you hear as a friend guesses what you drew. <br> 
Draw strategically as once a scribble is scribbled, it cannot be unscribbled!
